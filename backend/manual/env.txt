# Server
PORT=3000
NODE_ENV=development

# Database (PostgreSQL)
DATABASE_URL="postgresql://user:password@localhost:5432/openpoll?schema=public"

# Redis
REDIS_URL="redis://localhost:6379"

# JWT
JWT_SECRET=""
JWT_ACCESS_EXPIRES_IN="15m"
JWT_REFRESH_EXPIRES_IN="7d"

# CORS
CORS_ORIGIN="http://localhost:5173"

# admin 계정
ADMIN_PASSWORD="dkzkcmzl"

이게 기본 틀이고 신경 써야 될 부분이 DATABASE_URL이랑 JWT_SECRET임
(+npm install도 꼭 합시다...)

일단 DATABASE_URL 부분에는
user:password 저 부분에 user랑 password 채워주면 됨(그 DB 만들면서 넣었던 것들)
user는 보통 postgres로 다 할거고, password는 처음 SQL 설정 시에 넣는데 그거임
저처럼 로컬 말고 Docker에서 돌려도 어차피 백엔드가 로컬에서 돌기 때문에 저대로 하시면 됩니다
근데 만약에 이제 나중에 백엔드도 도커로 돌리면 그때는 postgres:5432로 해야 됨!
(모르겠으면 AI 선생님들께 여쭤보도록 합시다)

JWT_SECRET 자리에는
openssl rand -base64 64

node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
이렇게 node 써서 secret key 값을 뽑아서 쓰면 됨
